# TaskFlow
TaskFlow — це спрощена система керування задачами, створена за мотивами Trello, але без зайвого функціоналу, складної реєстрації та підписок.

## Технологіі
- HTML
- CSS
- JS

## Як користуватися?
Завантажте zip-архів і розпакуйте його в будь-яку зручну папку. Відкрийте цю папку за допомогою VSC або будь-якої іншої програми для редагування коду. Запустіть HTML-скрипт.
